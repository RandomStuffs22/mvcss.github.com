Basics

  • Alphabetize properties
  • Use soft-tabs with two-space indent
  • Do not use ids (only classes)
  • Put spaces after :
  • Put spaces after the // for comments
  • Do not put a space between rule declarations

      // This is an example!
      .example
        background: #FFF
        color: #000
        font-size: 1.6em
      
      .example-alt
        background: #000
        color: #FFF
        font-size: 2em
